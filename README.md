# 合规强采集工具 MVP

这是一个本地运行的合规采集工具首版。它不会实现验证码破解、IP 池绕封、指纹伪装、反检测脚本或绕过登录/付费墙等能力；遇到验证码、登录墙、403/429、频繁失败等访问控制或反滥用信号时，会暂停该来源并生成诊断资料，等待人工处理。

## 运行

```powershell
node src/server.js
```

然后打开：

```text
http://127.0.0.1:4173
```

## 已实现能力

- 本地 Web 管理界面
- 创建采集任务，配置来源 URL、字段模板、限速、超时、重试
- HTTP 采集引擎
- 403/429、验证码、登录墙、频繁失败自动暂停
- 保存 HTML 快照、文本截图占位、请求上下文、错误报告
- 人工处理后恢复任务
- 本地文件库导出 JSON/CSV
- 审计日志
- robots.txt 提示

## 安全边界

本项目只做授权范围内的稳定采集与诊断，不包含任何规避对方访问控制或反滥用机制的功能。
